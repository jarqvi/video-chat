
# Video-Chat

This is a simple video chat with socket.io


## Setting up

#### Cloning project

```http
  git clone https://github.com/jarqvi/video-chat.git
  cd video-chat
```
## Start

To run project

```bash
  npm run start
  npm run peer
```
